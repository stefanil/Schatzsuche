Schatzsuche

Hier entsteht das interaktive Browserspiel "Schatzsuche".

Der Spieler bewegt sich darin auf einer Landkarte, die das Dreiländereck zwischen Deutschland, Polen und der Tschechischen Republik umfasst. Er kann dabei zahlreiche Sehenswürdigkeiten entdecken und bekommt Informationen zu den besuchten Orten.
